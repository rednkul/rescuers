Руководство пользователя

  1. Общие рекомендации по работе с приложением
    • Первая буква в названии должности, подразделения, службы – всегда заглавная;
    • Между словами при именовании должностей, подразделений, служб – 1 пробел;
    • Перед скачиванием отчетов добавить хотя бы по одному подразделению, должности, службу штату должности в подразделении, сотруднику.

    Эти рекомендации помогут уменьшить количество ошибок работы приложения.

  2. Функции приложения

    Данное приложение осуществляет имеет функции:

    1. Добавление/редактирование/удаление подразделений, служб, должностей, штатных размеров должностей в подразделениях, сотрудников и вакансий;
    2. Получение excel-отчетов об укомплектованности отряда, списка сотрудников отряда и списка сотрудников, которым предстоит аттестация в ближайшие 90 дней.

    3. Уровни и права доступа

      Для действий, описанных в пункте 2, предусмотрены группы пользователей с различными правами доступа:

    • Уровень 0: имеет права доступа для всех действий;
    • Уровень 1: может добавлять/редактировать/удалять сотрудников и вакансии, скачивать отчеты;
    • Уровень 2: не имеет прав добавления/редактирования/удаления, только получение отчетов.

  4. Пояснения к страницам
    1. Домашняя страница. (‘Сотрудники’ в правом верхнем углу)

      После авторизации пользователь видит домашнюю страницу, которая представляет собой таблицу сотрудников отряда, разбитую по подразделениям

      В конце подтаблицы каждого подразделения выводится итоговая информация по подразделению, а под самой таблицей можно увидеть итоговую информацию по отряду.

      Внутри подразделения сотрудники выводятся соответственно приоритету, указанному при добавлении сотрудника/вакансии.

      Красным шрифтом указаны вакансии, желтым фоном – сотрудники, которым в ближайшие 90 дней предстоит переаттестация.

      Слева от таблицы находится блок с полем поиска по ФИО сотрудника и фильтрами по подразделению, должности, принадлежности к оперативному составу, принадлежности к спасателям, полу и по принадлежности к числу скороаттестующихся.

      Для скачивания списка скороаттестующихся нужно нажать на кнопку 'Скачать’ справа от соответствующего фильтра.

      Для выполнения поиска по ФИО после введения информации в поле нужно нажать на иконку лупы, для поиска по фильтрам – нажать на кнопку 'Найти’ под фильтрами.

      При выполнении поиска по ФИО/фильтрам итоговые таблицы подразделений и отряда будут скрыты, вместо них появится строка ‘Итого по запросу’ с количеством сотрудников, соответствующих выполненному запросу.

      Под фильтрами имеется несколько кнопок, ведущих на соответствующие страницы добавления.

      Для получения списка сотрудников в виде excel-файла необходимо нажать кнопку ‘Скачать список сотрудников’, находящуюся под итоговой таблицей отряда.

      При нажатии на имя сотрудника в таблице пользователь перейдет на страницу сотрудника, содержащую информацию о нем и фото. Под ними будут кнопки для редактирования и удаления сотрудника.

      При нажатии на название должности в таблице откроется страница с информацией о должности и списком штатных размеров этой должности в подразделениях отряда. Для редактирования/удаления должности нужно нажать на ее название в верхней ячейке таблицы.

    2. Страница подразделений (‘Подразделения’ в правом верхнем углу)

      Содержит список подразделений, при нажатии на любую из них будет совершен переход на страницу подразделения.

      Страница подразделения соответствует главной странице с несколькими отличиями:

      1. Нет фильтра по подразделению, а поиск по фильтрам и ФИО совершается внутри выбранного подразделения;

      2. Отсутствуют кнопки для скачивания отчетов;

      3. Нажатие на должность ведет на страницу штата(размера) этой должности в этом подразделении с возможностью его редактирования;

      4. Нажатие на название подразделения ведет на страницу редактирования подразделения.

    3. Страница укомплектованности отряда (‘Укомплектованность’ в правом верхнем углу)

      Содержит таблицу, примерно соответствующую стандартизированному отчету об укомплектованности. Под таблицей расположена кнопка для скачивания excel-отчета стандартного вида.

      Справа от таблицы находятся кнопки для добавления служб и должностей.

      При нажатии на службу в таблице откроется страница редактирования/удаления службы.

      При нажатии на название должности в таблице откроется страница с информацией о должности и списком штатных размеров этой должности в подразделениях отряда. Для редактирования/удаления должности нужно нажать на ее название в верхней ячейке таблицы.

    4. Добавление информации

      Чтобы информация в таблицах и отчетах выводилась корректно, необходимо соблюдать следущий порядок добавления информации:

      1. Добавить подразделения;
      2. Добавить службы;
      3. Добавить должности (для каждой должности указать службу);
      4. Добавить должности в подразделения;
      5. Добавить сотрудников/вакансии.
    5. Редактирование информации

      Редактировать информацию о подразделении можно по нажатию на соответсвующую кнопку на странице подразделения.

      Редактировать информацию о службе можно по нажатию на название службы на странице укомплектованности.

      Редактировать информацию о должности можно по нажатию на название сдолжности в таблице на домашней странице или странице подразделения.

      Информацию о должности в подразделении можно редактировать, перейдя на страницу соответствующего подразделения и нажав на название нужной должности.

      Информацию о сотруднике/вакансии можно редактировать по нажатию на имя сотрудника/вакансию в таблице на домашней странице или странице подразделения.

      При найме сотрудника необходимо перейти на страницу соответсвующей вакансии, нажать ‘Редактировать’. Если сотруднику требуется аттестация, после ее прохождения выбрать ‘ДА’ на пункте ‘Аттестован’ и указать дату аттестации.

    6. Удаление информации

      Кнопки для удаления объектов располагаются рядом с кнопками их редактирования.
